List is an interface from java.util package

List is an ordered collection
It allows duplicate elements to be added

There can be two types of list:
1. ArrayList
2. LinkedList

List/ArrayList/LinkedList methods:
1. add() - add element 
2. get() - return element at specific index
3. set() - replace element at specific index
4. remove() - remove element
5. size() - number of element in a list


LinkedList methods:
1. addFirst()
2. addLast()
3. getFirst()
4. getLast()
5. removeFirst()
6. removeLast()
